About Word Zest

Find out how quickly you can guess 10 random 5 letter words. You will have up to 6 guesses for each word.

Word Fun

Word Zest is an iPhone App based on the popular theme of word guessing. Although it is enjoyable to solve one word at a time, the main objective with Word Zest is to guess 10 consecutive words correctly in the quickest time possible. The game has several innovative features to assist in the pursuit for the fastest time. It also contains other unique features to provide a pleasurable user experience.

Favorite Words

You can save up to 5 favourite words (numbered 1 through 5). The game keypad has a number key used to play your favorite words. Use it so you don't have to type your favorite words. You will save time with this feature.

Autoplay Favourite Words

  • You can choose to Autoplay 1 or more favourite words.
  • With this setting, when a new game is started the favourite word or words will be automatically entered for you. For example, If you like to start each game with “AISLE”, specify it as favourite word #1 and choose Autoplay 1 word. When a new game is started “AISLE” will be entered for you.

Shake It Up!

Word Zest detects and acts upon shakes as follows:

  • If there are letters in the current row, shake to clear the row.
  • If the current row is blank, shake to enter the next favourite word.
  • If the game is over, shake to start the next game.
  • If you are not on the main view, shake to go back. For example, if you are looking at the statistics, shake to go back to the main view.

Game board is a Keypad

you can tap on any of the letters in the rows above the current row. This feature has two great benefits:

  • You can focus on the previously played letters while choosing the letters for your next word
  • The larger letters on the game board (compared to the keypad) can be tapped quicker and more accurately.

Get Encouraged!

Each time you correctly guess a word several things will happen to provide encouragement to the play:

  • A short message will flash showing how many consecutive games you have one and encourage you to get to 10 consecutive games.
  • A victory sound will be played. A more elaborate victory sound will be played if you have correctly guessed 10 or more words.
  • Confetti will be emitted from the top of the screen.
  • The colour of the icons for “Game Times” and “Statistics” will change to green, or gold (if you have correctly guessed 10 or more words).

Get Inspired!

  • The “Game Times” view shows a bar chart of the time taken to correctly guess each of the most recent 10 words. If you keep going after getting to 10 words, the time for each new word will be added to the top of the list and the oldest correctly guessed word will be pushed off the bottom of the list.
  • The “Game Times” view includes a graph for the number of guesses taken for each word. Each time you guess a word quicker than the last word on the list your 10-game time will improve.
  • There is also a graph for the scrabble points value for each guessed word. These graphs are included as eye candy to make playing the game more fun.

Stats Fun!

The Statistics view includes a bar chart and a pie chart showing how many guesses you have taken guessing the words.
